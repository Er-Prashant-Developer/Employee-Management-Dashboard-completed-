# Employee Management Dashboard (React)

A modern and responsive Employee Management Dashboard built using React, Context API, and LocalStorage.  
This application allows users to efficiently manage employee records with features like authentication, CRUD operations, search, filtering, and printing.

---

## Authentication

- Simple login system  
- Protected routes implemented  
- Only authenticated users can access the dashboard  

---

## Dashboard Overview

- Total Employees  
- Active Employees  
- Inactive Employees  

---

## Employee Management

- Add new employees  
- Edit existing employee details  
- Delete employees with confirmation  
- Upload employee profile image  

---

## Employee List

- View all employees in a table  
- Search by name  
- Filter by gender and status  

---

## Print Feature

- Print employee list using browser print  

---

## UI / UX

- Responsive design  
- Tailwind CSS  
- Mobile friendly  

---

## Data Storage

- LocalStorage  
- Context API for state management  

---

## Tech Stack

- React  
- React Router  
- Context API  
- Tailwind CSS
